# Атлас Антарктики, 
изданный в 1966 году, содержит данные, собранные во время и после Международного геофизического года, и предоставляет доступ к уникальным историческим наблюдениям.

# The Atlas of Antarctica, 
published in 1966, contains data collected during and after the International Geophysical Year and provides access to unique historical observations.
